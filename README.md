# Meshak Portfolio

A premium dark portfolio for Meshak, a Portugal-based BCA student, Frontend Developer, and AI Website Builder.

The site presents a clean modern profile, current coding skills, beginner-friendly projects, and a slightly futuristic visual style built around HTML, CSS, JavaScript, C++, Python, React, Next.js, Tailwind CSS, and motion.

## Focus

- Modern business websites
- BCA student projects
- HTML, CSS, JavaScript, C++, and Python
- React and Next.js interfaces
- Tailwind CSS
- AI tools and automation ideas
- Supabase, n8n, and GitHub

## Projects Featured

- Personal Portfolio Website
- Local Business Website Demo
- Restaurant Lead Generation Automation
- AI Workspace / SaaS Dashboard Concept

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
